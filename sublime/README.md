# sublime配置文件
## 使用方式(Linux)
添加本地配置文件:

```
// sublime配置文件
ln -s /opt/workspace/github/myconf/sublime/Preferences.sublime-settings ~/.config/sublime-text/Packages/User/Preferences.sublime-settings

// sublime快捷键配置
ln -s /opt/workspace/github/myconf/sublime/Default_Linux.sublime-keymap  ~/.config/sublime-text/Packages/User/Default\ \(Linux\).sublime-keymap
```
